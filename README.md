# node-test
函数式编程、文件上传、node 起步 demo

还有 express + mongoose 的一个用户 CRUD

## Project setup
```
npm install
```

## run for development
```
npm run dev
```
express will run on port 3001

native will run on port 3000

## accessable url

### native (port 3000)
/start

/upload

/show
/asynctest

### express (port 3001)
/user
